- Marquer toute incertitude juridique avec **[À VÉRIFIER]**.

## Section "Flashback" (répétition espacée)

En plus du rapport hebdomadaire, inclure une section Flashback :

1. Sélectionner aléatoirement 3 articles wiki créés il y a plus de 2 semaines.
2. Pour chacun, poser une question de vérification rapide (type quiz).
3. Présenter les questions à l'utilisateur et attendre ses réponses.
4. Si l'utilisateur ne sait plus ou se trompe → marquer l'article avec le tag
   `needs-review` dans son frontmatter pour qu'il revienne dans le prochain quiz.
5. Si l'utilisateur répond correctement → noter la date de dernière révision.

Format du flashback :
```
## 🔁 Flashback — Révision espacée

### Article 1 : [[nom-de-l-article]]
Créé le : YYYY-MM-DD | Dernière révision : YYYY-MM-DD
❓ Question : [question ciblée sur le contenu de l'article]

### Article 2 : [[nom-de-l-article]]
Créé le : YYYY-MM-DD | Dernière révision : YYYY-MM-DD
❓ Question : [question ciblée sur le contenu de l'article]

### Article 3 : [[nom-de-l-article]]
Créé le : YYYY-MM-DD | Dernière révision : YYYY-MM-DD
❓ Question : [question ciblée sur le contenu de l'article]
```

Objectif : éviter l'accumulation de 200 articles jamais relus. Forcer la
révision régulière des connaissances anciennes.
